AI_yoga_trainer:

This task is to AI yoga Trainer that detect Tadasana train in real time. MediaPipe was used to extract the main points in the body and then train the neural network on them so that it can recognize whether the movement is correct or wrong.

Dependencies:

python
tensorflow
opencv-python
mediapipe
scikit-learn
matplotlib

Installing dependencies:
install requirment.txt by run this command in cmd:
 
 pip install -r requirment.txt
