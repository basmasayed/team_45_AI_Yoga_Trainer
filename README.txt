AI_yoga_trainer:

This task is to AI yoga Trainer that detect Tadasana train in real time. MediaPipe was used to extract the main points in the body and then train the neural network on them so that it can recognize whether the movement is correct or wrong.
if you want to retrain the model you should download this dataset and extract it beside AI_yoga_trainer.ipynb or any dataset you want:
https://drive.google.com/file/d/1fRVLKorLmD3rw6tyC5KvEamqzyuHyUzG/view?usp=sharing

Dependencies:

python
tensorflow
opencv-python
mediapipe
scikit-learn
matplotlib

Installing dependencies:
install requirment.txt by run this command in cmd:
 
 pip install -r requirment.txt
